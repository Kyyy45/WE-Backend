## LOGIN (Cookie saved ‚Üí cookie.txt)

curl -c cookie.txt -X POST [http://localhost:5000/api/v1/auth/login](http://localhost:5000/api/v1/auth/login) 
-H "Content-Type: application/json" 
-d '{"emailOrUsername":"[teacherworldpedia888@yopmail.com](mailto:teacherworldpedia888@yopmail.com)","password":"TeacherPassw0rd!"}'

## REFRESH TOKEN

curl -b cookie.txt -X POST [http://localhost:5000/api/v1/auth/refresh-token](http://localhost:5000/api/v1/auth/refresh-token)

## LOGOUT

curl -b cookie.txt -X POST [http://localhost:5000/api/v1/auth/logout](http://localhost:5000/api/v1/auth/logout)

## FORGOT PASSWORD

curl -X POST [http://localhost:5000/api/v1/auth/forgot-password](http://localhost:5000/api/v1/auth/forgot-password) 
-H "Content-Type: application/json" 
-d '{"email":"[studentworldpedia888@yopmail.com](mailto:studentworldpedia888@yopmail.com)"}'

## RESET PASSWORD

curl -X POST [http://localhost:5000/api/v1/auth/reset-password](http://localhost:5000/api/v1/auth/reset-password) 
-H "Content-Type: application/json" 
-d '{"email":"[studentworldpedia888@yopmail.com](mailto:studentworldpedia888@yopmail.com)","token":"<token>","newPassword":"NewPassw0rd!"}'

üë§ PROFILE MANAGEMENT
=====================

## GET PROFILE (Logged user)

curl -H "Authorization: Bearer <ACCESS_TOKEN>" 
[http://localhost:5000/api/v1/profile](http://localhost:5000/api/v1/profile)

## UPDATE PROFILE (username / avatarUrl)

curl -X PUT [http://localhost:5000/api/v1/profile](http://localhost:5000/api/v1/profile) 
-H "Authorization: Bearer <ACCESS_TOKEN>" 
-H "Content-Type: application/json" 
-d '{"username": "UpdatedTeacherWE"}'

## CHANGE PASSWORD

curl -X PUT [http://localhost:5000/api/v1/profile/password](http://localhost:5000/api/v1/profile/password) 
-H "Authorization: Bearer <ACCESS_TOKEN>" 
-H "Content-Type: application/json" 
-d '{"oldPassword":"TeacherPassw0rd!","newPassword":"TeacherPassw0rdNEW!"}'

## UPLOAD AVATAR

curl -X PUT [http://localhost:5000/api/v1/profile/avatar](http://localhost:5000/api/v1/profile/avatar) 
-H "Authorization: Bearer <ACCESS_TOKEN>" 
-F "avatar=@./sample-avatar.jpg"

üìö COURSE MANAGEMENT
====================

## CREATE COURSE (Admin only)

curl -X POST [http://localhost:5000/api/v1/courses](http://localhost:5000/api/v1/courses) 
-H "Authorization: Bearer <ACCESS_TOKEN_ADMIN>" 
-H "Content-Type: application/json" 
-d '{
"title": "Letterland",
"shortDescription": "Kursus membaca interaktif",
"description": "Pelajari membaca interaktif untuk anak usia dini",
"teacherId": "<TEACHER_ID>",
"price": 0,
"category": "Literacy",
"isPublished": true
}'

## LIST COURSES

curl -H "Authorization: Bearer <ACCESS_TOKEN>" 
[http://localhost:5000/api/v1/courses](http://localhost:5000/api/v1/courses)

## GET COURSE BY ID / SLUG

curl -H "Authorization: Bearer <ACCESS_TOKEN>" 
[http://localhost:5000/api/v1/courses/](http://localhost:5000/api/v1/courses/)<COURSE_ID_OR_SLUG>

## UPDATE COURSE (Admin)

curl -X PUT [http://localhost:5000/api/v1/courses/](http://localhost:5000/api/v1/courses/)<COURSE_ID> 
-H "Authorization: Bearer <ACCESS_TOKEN_ADMIN>" 
-H "Content-Type: application/json" 
-d '{"title":"Letterland Advanced","price":150000}'

## DELETE COURSE (Admin)

curl -X DELETE [http://localhost:5000/api/v1/courses/](http://localhost:5000/api/v1/courses/)<COURSE_ID> 
-H "Authorization: Bearer <ACCESS_TOKEN_ADMIN>"
